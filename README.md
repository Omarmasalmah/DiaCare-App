# 🩺 DiaCare – Diabetes Support & Prediction App

*A smart assistant app for diabetes prediction and daily management using ML and Flutter.*

---

## 📱 Features

| Feature                          | Description                                                                 |
|----------------------------------|-----------------------------------------------------------------------------|
| 🔍 **Diabetes Prediction**       | ML-based prediction of diabetes risk based on user health inputs.          |
| 🥗 **Insulin Dose Calculator**   | Calculates insulin dosage based on food intake and nutritional info.       |
| 🏃 **Home-Based Workouts**       | Exercise recommendations tailored for diabetic users.                      |
| 📍 **Nearby Centers**            | Integrated Google Maps to locate the nearest diabetes support centers.     |
| 📈 **Glucose Tracking**          | Visualizes blood glucose levels over time with interactive charts.         |
| 💬 **Doctor Chat**               | Real-time chat with healthcare professionals using Firebase.               |
| 🌐 **Community Feed**            | Share tips, experiences, and motivation among users and doctors.           |

---

## 🤖 Machine Learning Workflow

- Data preprocessing and feature engineering on diabetes datasets.
- Applied balancing techniques for class imbalance.
- Trained and evaluated multiple models using:
  - Accuracy
  - Precision
  - Recall
  - F1-Score
- Final model deployed using a **Flask API** and integrated with the app.
